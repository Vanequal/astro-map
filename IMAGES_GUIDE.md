# Руководство по изображениям для AstroMap

## Где размещать изображения

Все изображения должны быть помещены в папку:
```
src/assets/images/
```

## Список необходимых изображений

### Основные элементы UI
- **logo.svg** - Логотип сайта (используется в хедере)
- **burger-menu.svg** - Иконка бургер-меню
- **star.svg** - Маленькая звездочка (используется часто)
- **star.png** - Большая звездочка
- **settings.svg** - Иконка настроек

### Профиль
- **avatar-profile.png** - Аватар пользователя (используется в бургер-меню)

### Фоновые изображения
- **main-bg.png** - Фон для hero секции (космос/звезды)
- **main-2-bg.png** - Фон для about секции
- **main-bg-5.png** - Фон для subscription секции
- **main-bg-7.png** - Фон для astrologer секции

### Карточки и иконки
- **sun-card.svg** - Иконка солнца (Работа и доход)
- **love-card.svg** - Иконка сердца (Отношения и любовь)
- **health-card.svg** - Иконка здоровья
- **life-card.svg** - Иконка жизненного пути

### Модули курса
- **module-1.png** - Модуль 1: Введение в астрологию
- **module-2.png** - Модуль 2: Планеты
- **module-3.png** - Модуль 3: Знаки Зодиака
- **module-4.png** - Модуль 4: Астрологические дома
- **module-5.png** - Модуль 5: Аспекты между планетами
- **module-6.png** - Модуль 6: Построение и чтение карты

### Дополнительные изображения
- **astrologic-circle.png** - Астрологический круг/колесо
- **profesional.png** - Изображение для секции профессий
- **partner.png** - Изображение для секции партнеров
- **modal-2.png** - Правая часть модального окна подписки
- **cta.png** - Изображение для финальной CTA секции

## Рекомендации по размерам

### Логотип и иконки
- logo.svg: высота ~50px
- Иконки (star, settings и т.д.): 30-60px

### Фоновые изображения
- Рекомендуемый размер: 1920x1080px или больше
- Формат: PNG или JPG
- Оптимизировать для веба

### Карточки
- Иконки карточек: 60-80px
- Изображения модулей: 80-100px

### Большие изображения
- astrologic-circle.png: ~400px ширина
- profesional.png, partner.png: ~450px
- cta.png: ~500px
- modal-2.png: высота под модальное окно (~600px высота)

## Форматы файлов

- **SVG** - для иконок и простых графических элементов
- **PNG** - для изображений с прозрачностью
- **JPG** - для фотографий и сложных изображений без прозрачности

## Оптимизация

Рекомендуется оптимизировать все изображения перед использованием:
- SVG: удалить лишние метаданные
- PNG/JPG: сжать без потери качества (TinyPNG, ImageOptim и т.д.)

## Плейсхолдеры

Если у вас пока нет некоторых изображений, можно:
1. Использовать временные плейсхолдеры
2. Создать простые SVG иконки
3. Использовать https://placeholder.com/ для временных изображений

Пример пути в коде:
```jsx
<img src="/src/assets/images/logo.svg" alt="AstroMap Logo" />
```
