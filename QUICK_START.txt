📦 ЧТО УЖЕ СДЕЛАНО:

✅ Чистая верстка на React + Vite
✅ БЭМ методология для CSS
✅ Все 10 секций главной страницы
✅ Хедер с 2 состояниями (главная/внутренние)
✅ Бургер-меню с навигацией
✅ Футер
✅ 2 модальных окна (регистрация + подписка)
✅ FAQ с аккордеоном
✅ Семантичная разметка
✅ SEO оптимизация
✅ Все анимации и ховеры


🚀 КАК ЗАПУСТИТЬ:

1. Распакуйте архив
2. npm install
3. npm run dev
4. Откройте http://localhost:5173


📁 КУДА КИДАТЬ ИЗОБРАЖЕНИЯ:

Все ваши изображения поместите в:
   src/assets/images/

Список всех нужных файлов смотри в IMAGES_GUIDE.md


🎨 ШРИФТЫ:

Сейчас используются системные шрифты.
Если нужны кастомные:
- Положите в src/assets/fonts/
- Обновите @font-face в src/index.css


📄 СТРУКТУРА:

src/
├── components/
│   ├── Header/Header.jsx       # Хедер
│   ├── Footer/Footer.jsx       # Футер
│   └── Modal/                  # Модалки
├── pages/
│   └── Home.jsx                # Главная (все 10 секций)
├── index.css                   # Глобальные стили + БЭМ
└── Home.css                    # Стили главной страницы


🎯 ЧТО ДАЛЬШЕ:

1. Добавь изображения в src/assets/images/
2. При необходимости добавь шрифты
3. Создай остальные страницы (используй те же компоненты)
4. Подключи функционал (формы, API и т.д.)


💡 ЗАМЕТКИ:

- Все элементы следуют БЭМ: блок__элемент--модификатор
- Константы цветов можно вынести в CSS переменные
- Адаптив добавить по желанию (сейчас desktop-first)
- Можно добавить React Router для остальных страниц


Вопросы? Смотри README.md и IMAGES_GUIDE.md
